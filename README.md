Python code for raspberry pi pico to moderate air land and sea with lights

White = Air
Green = Land
Blue = Sea

Requirements
------------
* 1 x 16 RGB WS2812 Light Ring
* 3 M to M Jumper wires
* 1 Breadboard

Download source code
--------------------
```
$ gh repo clone maxkung101/airLandAndSeaLights
```
